# 🎵⚒️ GrooveForge\n\n[![GrooveForge CI](https://github.com/relix42/grooveforge/actions/workflows/ci.yml/badge.svg)](https://github.com/relix42/grooveforge/actions/workflows/ci.yml)\n[![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](https://opensource.org/licenses/Apache-2.0)\n[![npm version](https://badge.fury.io/js/%40relix42%2Fgrooveforge.svg)](https://badge.fury.io/js/%40relix42%2Fgrooveforge)\n\n![GrooveForge Screenshot](./docs/assets/grooveforge-screenshot.png)\n\n**🎵⚒️ Find your groove, forge your future with AI**\n\nGrooveForge is an independent, privacy-focused AI development companion that helps you find your coding rhythm and forge ideas into reality. Built for developers who value privacy, local control, and the freedom to choose their AI provider.\n\n## ✨ Why GrooveForge?\n\n🔒 **Privacy First** - No data collection, no telemetry, your code stays yours  \n🏠 **Local Control** - Run AI models locally with Ollama or choose your cloud provider  \n🎯 **Independent** - Not tied to any single AI company or platform  \n⚡ **Developer Focused** - Built by developers, for developers  \n🛠️ **Extensible** - MCP server support for unlimited customization  \n🎵 **Find Your Groove** - Intuitive workflow that adapts to your coding style  \n\n## 🚀 Quick Start\n\n### Installation\n\n```bash\n# Install globally via npm\nnpm install -g @relix42/grooveforge\n\n# Run from anywhere\ngrooveforge\n# or use the short alias\ngf\n```\n\n### Choose Your AI Provider\n\nGrooveForge supports multiple AI providers - choose what works best for you:\n\n#### 🦙 Ollama (Recommended - Local & Private)\n\n```bash\n# 1. Install Ollama\ncurl -fsSL https://ollama.ai/install.sh | sh\n\n# 2. Pull a model\nollama pull llama3.2\n\n# 3. Configure GrooveForge\nexport CHAT_CLI_PROVIDER=\"ollama\"\nexport OLLAMA_MODEL=\"llama3.2\"\n\n# 4. Start coding!\ngrooveforge\n```\n\n#### 🤖 Claude (Cloud)\n\n```bash\n# 1. Get your Claude API key from Anthropic\n# 2. Configure GrooveForge\nexport CHAT_CLI_PROVIDER=\"claude\"\nexport CLAUDE_API_KEY=\"your_api_key_here\"\n\n# 3. Start coding!\ngrooveforge\n```\n\n#### 💎 Gemini (Legacy Support)\n\n```bash\n# 1. Get your Gemini API key from Google AI Studio\n# 2. Configure GrooveForge\nexport GEMINI_API_KEY=\"your_api_key_here\"\n\n# 3. Start coding!\ngrooveforge\n```\n\n## 🎯 Core Features\n\n### 💬 Intelligent Code Assistance\n- **Natural Language Coding** - Describe what you want, get working code\n- **Codebase Understanding** - AI that understands your entire project context\n- **Smart Refactoring** - Modernize and improve existing code\n- **Bug Detection** - Find and fix issues before they become problems\n\n### 🔧 Powerful Slash Commands\n- `/providers` - Manage AI providers (Ollama, Claude, Gemini)\n- `/memory` - Persistent context across sessions\n- `/tools` - Extend capabilities with custom tools\n- `/mcp` - Model Context Protocol server integration\n- `/initial-prompts` - Automate common workflows\n- `/stats` - Track your productivity and usage\n\n### 🛠️ Developer Workflow Integration\n- **Git Integration** - Understand commit history and changes\n- **File Operations** - Read, write, and modify files intelligently\n- **Shell Commands** - Execute system commands with AI guidance\n- **Project Memory** - Remembers your project structure and preferences\n\n## 📖 Usage Examples\n\n### Start a New Project\n```bash\ncd my-new-project/\ngrooveforge\n> Help me create a REST API for a todo app using Node.js and Express\n```\n\n### Understand Existing Code\n```bash\ncd existing-project/\ngrooveforge\n> Analyze this codebase and explain the main architecture patterns\n```\n\n### Debug and Fix Issues\n```bash\ngrooveforge\n> I'm getting a \"Cannot read property 'map' of undefined\" error. Help me find and fix it.\n```\n\n### Refactor and Modernize\n```bash\ngrooveforge\n> Convert this JavaScript code to TypeScript with proper type definitions\n```\n\n## 🎵 Finding Your Groove\n\nGrooveForge adapts to your coding style and workflow:\n\n### Initial Prompts\nAutomate your startup routine:\n```bash\ngrooveforge --initial-prompt \"Review recent git commits\" --initial-prompt \"Check for TODO comments\"\n```\n\n### Memory System\nGrooveForge remembers your project context:\n```bash\n> /memory show  # See what GrooveForge remembers about your project\n> /memory refresh  # Update project understanding\n```\n\n### Provider Switching\nSwitch between AI providers seamlessly:\n```bash\n> /providers list     # See available providers\n> /providers current  # Check current provider\n> /providers switch ollama  # Switch to Ollama\n```\n\n## 🔧 Advanced Configuration\n\n### Environment Variables\n\n```bash\n# AI Provider Configuration\nCHAT_CLI_PROVIDER=ollama|claude|gemini\nOLLAMA_MODEL=llama3.2\nOLLAMA_URL=http://localhost:11434\nCLAUDE_API_KEY=your_claude_key\nCLAUDE_MODEL=claude-3-5-sonnet-20241022\nGEMINI_API_KEY=your_gemini_key\n\n# GrooveForge Settings\nGROOVEFORGE_THEME=dark|light\nGROOVEFORGE_MEMORY_ENABLED=true\nGROOVEFORGE_AUTO_SAVE=true\n```\n\n### Configuration File\nCreate `.grooveforge/settings.json` in your project or home directory:\n\n```json\n{\n  \"theme\": \"dark\",\n  \"selectedAuthType\": \"OLLAMA\",\n  \"memoryEnabled\": true,\n  \"autoSave\": true,\n  \"preferredEditor\": \"vscode\",\n  \"initialPrompts\": [\n    \"Review recent changes\",\n    \"Check for security issues\"\n  ]\n}\n```\n\n## 🏗️ Development Setup\n\n### Prerequisites\n- Node.js 20+ \n- npm or yarn\n- Git\n\n### Local Development\n\n```bash\n# Clone the repository\ngit clone https://github.com/relix42/grooveforge.git\ncd grooveforge\n\n# Install dependencies\nnpm install\n\n# Build the project\nnpm run build\n\n# Run locally\nnpm start\n# or use the enhanced startup script\n./gf.sh\n\n# Run tests\nnpm test\n\n# Lint and format\nnpm run lint\nnpm run format\n```\n\n### Project Structure\n\n```\ngrooveforge/\n├── packages/\n│   ├── cli/           #
